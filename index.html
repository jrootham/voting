Map Voting Data to Statscan Data

Project Description

Create a geospatial database mapping Statscan data to poll level voting data.

Applications

Redistricting

An app to allow experimenting with different district boundaries.

Campaign Models

An app to model campaigns, especially multi-member campaigns.
